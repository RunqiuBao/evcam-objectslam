### How to build:

- At current dir, run `python3 -m build` to build package.
- `cd dist/` and then `python3 -m pip install tool_eventdetectobjects-0.0.1-py3-none-any.whl`
  (if `tool_eventdetectobjects` was already installed, uninstall first by `python3 -m pip uninstall tool_eventdetectobjects`)
